# Medi-Bank (Python OOP Banking System)

## Medi-Bank Python Project

Medi-Bank is a simple banking system built using **Object-Oriented
Programming (OOP) in Python**. It allows a user to:

-   Create an account (Savings or Current)
-   Deposit money
-   Withdraw money
-   Check account balance
-   View transaction history (with timestamps)
-   Automatically save transactions to a text file

This project was built to practice: - Classes & Objects - Inheritance
(SavingsAccount, CurrentAccount) - Methods - File handling - Date & time
logging - Basic user input flow


## Features

### 1. Create an Account

User enters: - Name - Age - Occupation - Account type (Savings or
Current) - Opening balance

### 2. Deposit Money

-   User enters how much they want to deposit
-   Balance updates
-   Transaction recorded with date & time

### 3. Withdraw Money

-   Prevents withdrawing more than available
-   Transaction logged

### 4. Check Balance

Shows the current account balance.

### 5. View Transaction History

Displays all deposit and withdrawal activity. Also saves as a `.txt`
file inside `my_folder/`.
